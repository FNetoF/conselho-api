## Instalação
### Siga estas etapas (dentro de um terminal):
1. Instale às dependências:
    ```shell
    $ npm install
    ```
    | |

    ```shell
    $ yarn install
    ```

<br>

## Executando o projeto
1. Executar o projeto:
    ```shell
    $ npm run dev
    ```

    | |

    ```shell
    $ yarn dev
    ```
2. localhost:
    ```http
    http://localhost:3000
    ```
